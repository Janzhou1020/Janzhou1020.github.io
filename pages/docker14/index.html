Harbor 私有镜像仓库

安装部署

安装 docker

安装 docker-compose

下载 harbor 压缩包

cd /opt/soft/harbor

vim harbor.cfg

hostname = 10.0.0.88

harbor_admin_password = 123456

./install.sh

docker 从 harbor 上传下载镜像

先将仓库地址加入到 docker 收信任的列表

vi /etc/docker/daemon.json

systemctl daemon-reload

systemctl restart docker

{
  "registry-mirrors": [
    "https://hub.atomgit.com"
  ],
  "insecure-registries": [
    "10.0.0.88"
  ]
}
