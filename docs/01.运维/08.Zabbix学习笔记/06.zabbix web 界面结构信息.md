```
主机群组（Host Group）
└── 主机（Host）
    └── 模板（Template）
        ├── 监控项（Items）
        ├── 触发器（Triggers）
        ├── 图形（Graphs）
        └── 自动发现规则（LLD Rules）
```

应用集：将相关联的监控项归类到一起

例如，你可以将所有与 CPU 相关的监控项放入一个名为 "CPU" 的应用集中，将所有与内存相关的监控项放入 "Memory" 的应用集中。这样可以更清晰地组织和管理不同类别的监控项
