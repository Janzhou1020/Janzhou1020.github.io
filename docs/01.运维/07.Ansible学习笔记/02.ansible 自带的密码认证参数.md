可以在 host 文件中自定义一些信息

ansible_host：主机地址
ansible_port：端口，默认 22
ansible_user：认证用户
ansible_ssh_pass：用户认证的密码

```
[ceshi]
10.0.0.21 ansible_user=root ansible_ssh_pass=123456
10.0.0.22 ansible_user=root ansible_ssh_pass=123456
```
