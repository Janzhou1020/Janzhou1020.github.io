在 MySQL 中，常用的权限包括 SELECT、INSERT、UPDATE、DELETE、CREATE、DROP、GRANT OPTION 等

在不限制主机的情况下，可以根据实际情况替换 % 为主机 IP：CREATE USER 'kf'@'%' IDENTIFIED BY 'Hello88';

在不限制数据库的表的情况下，可以将 \*.\* 替换为特定的数据库和表名：GRANT SELECT ON \*.\* TO 'kf'@'%';
